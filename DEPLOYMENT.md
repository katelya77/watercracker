### 🚀 部署到GitHub Pages

项目已配置自动部署，当推送到main分支时会自动构建并部署到GitHub Pages。

部署地址: `https://katelya77.github.io/watercracker`

### 🛠️ 本地开发

```bash
# 安装依赖
npm install

# 开发模式
npm run dev

# 构建生产版本
npm run build

# 预览构建结果
npm run preview
```
